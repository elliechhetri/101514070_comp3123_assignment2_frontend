# COMP3123 Assignment 2 Frontend
Student ID: 101514070

React frontend for employee management system.
Uses:
- React
- Axios
- React Router DOM
- JWT stored in localStorage

Features:
- Signup
- Login
- Employee list
- Add employee
- Edit employee
- Delete employee
- Search by department or position

How to run:
npm install
npm start

Backend must run at:
http://localhost:5000/api
